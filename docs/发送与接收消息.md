## 发送与响应的消息格式
* 发送消息   `lib/protocol/sdkws/sdkws.proto  MsgData`
* server 推送的消息格式  `lib/protocol/sdkws/sdkws.proto PushMessages`

golang 推送的消息，data 是protobuf 编码的字节串，由于我改造为json编码， 这里要用base64解码下
